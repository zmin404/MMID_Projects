== Description ==

<strong>Unlimited</strong> WordPress Widget Areas / Sidebars! Easily create them using this wonderful plugin and automatically display those sidebars below or after post, pages and/or custom post types content without coding!

[vimeo https://vimeo.com/205341430]

In need to Widget Areas manually on your template files? PHP code is also provided to each Widget Area dashboard, just copy them and paste it to the area you want them to show!

You can also just paste the Widget Areas shortcodes to display them anywhere you want! Just use this plugin together with <strong><a href="https://widget-options.com/?utm_source=widget-areas-org">Extended Widget Options for WordPress</a></strong> and you will have better control over each widget areas and sidebars you've created. Simple!

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the full directory into your wp-content/plugins directory
2. Activate the plugin at the plugin administration page
3. Go to Appearance > Widget Areas and start creating your custom Widget Areas
4. Then go to Appearance > Widgets to start adding contents
5. I hope you'll love this plugin :)
